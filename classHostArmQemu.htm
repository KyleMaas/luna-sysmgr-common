Detailed Description

Device-specific functionality for the Qemu ARM emulator

Device details:

  • Requires a bunch of key remapping.
  • Switches (0)
  • Translates a bunch of desktop keys to hardware keycodes. See HostArmQemuKeyFilter for information on which keys are remapped to which phone keycodes.
